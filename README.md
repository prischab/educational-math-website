# educational-math-website
Interactive educational math website for children aged 5–10, combining gamified learning, musical feedback, and progress tracking.

## Overview
This project is an interactive math learning website designed to make early education more fun and engaging for children. It uses gamified activities, music-based feedback, visual rewards, and progress tracking to create a child-friendly learning experience. The platform also includes parent-focused features for monitoring learning progress.

## Features
- Interactive math practice for young children
- Child-friendly and visually engaging design
- Musical feedback for better engagement
- Reward-based learning system
- Progress tracking and score display
- Parent dashboard and progress charts
- Learning recommendations
- Animated characters and playful UI elements

## Tech Stack
- Next.js
- React
- TypeScript
- CSS

## Project Structure
- `app/` – main application pages
- `components/` – reusable UI and feature components
- `hooks/` – custom React hooks
- `lib/` – utility functions
- `public/` – static assets and images
- `styles/` – styling files

## Main Components
- `math-problem.tsx` – displays math questions
- `music-player.tsx` – handles audio feedback
- `progress-tracker.tsx` – tracks child performance
- `reward-system.tsx` – manages rewards and gamification
- `score-display.tsx` – shows scores
- `parent-chart.tsx` – displays progress visually for parents
- `learning-recommendations.tsx` – suggests next learning steps
- `animated-character.tsx` – adds engaging animated visuals

## Installation
Install dependencies:

npm install

or

pnpm install

## Run the Project
Start the development server:

npm run dev

or

pnpm dev
